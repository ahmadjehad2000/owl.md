SQLite Schema — db.sqlite

All tables live in a single .owl/db.sqlite file. The markdown files are authoritative; this database is a derived index that can be fully rebuilt from the vault.

notes_fts MVP
SQLite FTS5 virtual table — mirrors notes(title, content).
Auto-updated via triggers on insert/update/delete.
Supports: MATCH 'phrase', prefix search, BM25 ranking.
Query: SELECT * FROM notes_fts WHERE notes_fts MATCH ?
ORDER BY bm25(notes_fts, 10, 1) — title hits weighted 10×.
