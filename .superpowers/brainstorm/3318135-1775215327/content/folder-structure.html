Key invariants

• Notes are plain .md files — readable without owl.md
• .owl/ is derived state — deletable and rebuildable
• SQLite is the only process that writes db.sqlite
• Filesystem watcher triggers incremental reindex on external edits
• Vault path is user-chosen at open time — no hardcoded locations
• config.json stores vault name, created date, schema version
